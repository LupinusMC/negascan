# シナリオ・デバフ因子集計ツール ウマ娘プリティーダービーの因子周回で手に入れた「デバフ因子」と「シナリオ因子」を記録・集計し、継承親の組み合わせを管理するためのツールです。 殿堂入りしたウマ娘（親）ごとに、親本人・祖父母の持つ因子の星の数を記録しておくことで、どの親の組み合わせが最も多くの因子を継承できるかを比較・検討することができます。 --- ## 主な機能 ### 親管理タブ 「親管理」タブでは、殿堂入りしたウマ娘の因子情報を登録・管理します。 #### 親の新規登録 1. **キャラ名・トレーナー名**を入力します。 2. **登録日**を入力します。数字を6桁（例: 260228）入力すると、自動的に「26/02/28」の形式にフォーマットされ、次の入力欄にカーソルが移動します。 3. **ダート・マイル**（赤因子）の星の数を入力します。0〜3の数字を入力すると、自動的に次のマスにカーソルが移動します（爆速入力）。値がない場合は0を入力するか、空欄のまま次へ進んでください。 4. **各因子の星の数**を同様に入力していきます。表示されているすべての因子について、親・祖1・祖2の3行分を入力します。 5. 入力が完了したら「親を登録」ボタンを押して保存します。 #### 祖の読込 祖父母の因子を手入力する代わりに、既に登録済みの親データを「祖1」「祖2」として読み込むことができます。プルダウンから該当する親を選択すると、そのデータの親1行目の因子が自動的にコピーされます。 #### 編集・削除 親一覧に表示されている各データの「編集」ボタンを押すと、そのデータが上部の入力欄に読み込まれ、内容を修正して「上書き保存」することができます。「削除」ボタンで登録を削除できますが、編成で使用中の親は先に編成を削除する必要があります。 ### 編成管理タブ 「編成管理」タブでは、登録済みの親を2人選んで「編成」として保存し、6人分（親1の親・祖1・祖2 ＋ 親2の親・祖1・祖2）の因子合計を確認することができます。 1. **親1・親2**をプルダウンから選択します。 2. 任意で**編成名**を入力します（未入力の場合は「名無し編成」になります）。 3. 「編成を保存」ボタンで保存します。 保存した編成は一覧に表示され、「詳細を表示」ボタンで6人分の因子の内訳を展開して確認できます。編成名は後から「名前変更」ボタンで変更することも可能です。 ### 表示フィルター 画面上部の「表示フィルター」から、因子のグループ（駆け引き、けん制、焦り、ためらい、点火、シナリオ他、真髄、加護）ごとに表示・非表示を切り替えることができます。 - チェックを外したグループの因子は、テーブルの列ごと非表示になります。 - 「星数計」「因子種類数」の値もフィルターに連動し、**表示中の因子のみ**を対象に計算されます。 - 並び替えのスコアもフィルターに連動します。例えば「ためらい」だけにチェックを入れて並び替えれば、ためらい因子だけで最も優秀な親を探すことができます。 ### 並び替え機能 「親一覧」および「編成一覧」の横にある「並び替え」プルダウンから、以下の基準でデータを並び替えることができます。 - **登録順**: データが登録された順番（デフォルト）で表示します。 - **星数計（多い順）**: 表示中のフィルターに基づく星の合計数が多い順に並び替えます。 - **因子種類数（多い順）**: 表示中のフィルターに基づく因子の種類数（ユニーク数）が多い順に並び替えます。 ### データの保存と管理 - **自動保存**: 登録・編集・削除の操作は、すべてブラウザのローカルストレージに自動的に保存されます。ページを閉じてもデータは保持されます。 - **JSON保存**: 「💾 JSON保存」ボタンを押すと、すべてのデータ（親データ＋編成データ）をJSONファイルとしてダウンロードできます。バックアップや端末間の移行にご利用ください。 - **JSON読込**: 「📂 JSON読込」ボタンを押すと、保存したJSONファイルからデータを復元できます。 --- ## 注意事項 - データはブラウザのローカルストレージに保存されます。**ブラウザのデータ消去（キャッシュクリア等）を行うと、因子データも一緒に消えてしまいます**。定期的に「JSON保存」でバックアップを取ることを強くおすすめします。 - 「全データ削除」ボタンを押すと、親データと編成データがすべて削除されます。この操作は元に戻せませんのでご注意ください。 ## 対応している因子 | カテゴリ | 因子 | |---------|------| | デバフ | けん制、焦り、ためらい（逃げ/先行/差し/追込の各4種） | | 駆け引き | 逃げ/先行/差し/追込 駆け引き | | 適性（赤因子） | ダート、マイル | | 点火 | 速、体、力、根、賢 | | 真髄 | 速、体、力、根、賢、心 | | 加護 | 陽、海、地 | | シナリオ他 | 綺羅星、夢の途中、前だけ見据えて、想いを背負って、バーニングソウル、いつまでも健やかに、夢の再生方法、時代を変える者、あるがままに、本気で休んで、アメリカン | ## 動作環境 本ツールは**PC（パソコン）での利用を推奨**しています。スマートフォンからも閲覧・操作は可能ですが、テーブルの横幅が広いため、PCの方が快適にご利用いただけます。 ## お問い合わせ ご要望・バグ報告等がありましたら、X（旧Twitter）アカウント [@brick29rs](https://x.com/brick29rs) までお願いします。
